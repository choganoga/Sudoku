About the game: A traditional Sudoku is a 9x9 puzzle grid made up of nine 3x3 regions. Each region, row, and column contains nine cells each. The game starts with an arbitrary amount of given numbers on the grid. The puzzle is solved by filling in the rest of the cells in the grid with the correct numbers based on the following rules:
  • Every number is between 1 and 9
  • Each row, column, and region will contain all 9 numbers, exactly once
There are many different techniques one can use to solve the puzzle. Some of these techniques are outlined here and here. No math is involved in solving the puzzle; it is purely a logic-based game.

Description of game logic: Given numbers are in a different style than user-entered numbers, and preferably given numbers cannot be changed. The player can get feedback on incorrect numbers at any time. There are multiple difficulty levels and multiple puzzles to choose from in each difficulty level. Puzzles are loaded using Ajax. There is a timer that tells how long the player has been working on the puzzle and a list of high scores based on the time required to complete the puzzle.
